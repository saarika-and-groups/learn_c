Data Types in R

R supports various data types, including numeric, character, logical, and complex. Understanding these data types is crucial for effective data manipulation and analysis.

  • Numeric: Represents numbers, both integers and floating-point.
  • Character: Represents text strings.
  • Logical: Represents boolean values (TRUE or FALSE).
  • Complex: Represents complex numbers with real and imaginary parts.
  • Factor: Represents categorical data with levels.
  • Data Frame: A table-like structure that can hold different data types in columns.
  • List: A collection of objects that can be of different types.
  • Matrix: A two-dimensional array that holds elements of the same data type.
  • Array: A multi-dimensional structure that holds elements of the same data type.
            # Example of numeric and character data types
            my_numeric <- 42
            my_character <- "Hello R"
            print(my_numeric)
            print(my_character)

Output:

[1] 42
[1] "Hello R"

Numeric

Decimal values are called numeric in R. It is the default R data type for numbers in R. If we assign a decimal value to a variable x as follows, x will be of numeric type. Real numbers with a decimal point are represented using this data type in R. It uses a format for double-precision floating-point numbers to represent numerical values.

            x = 5.6

            print(class(x))

            print(typeof(x))
[1] "numeric" [1] "double"

Interger

R supports integer data types which are the set of all integers. we can create as well as convert a value into an integer type using the as.integer() function. we can also use the capital 'L' notation as a suffix to denote that a particular value is of the integer R data type. 

x = as.integer(5)

print(class(x))

print(typeof(x))

y = 5L

print(class(y))

print(typeof(y))

Logical

R has logical data types that take either a value of true or false. A logical value is often created via a comparison between variables. Boolean values, which have two possible values, are represented by this R data type: FALSE or TRUE 

x = 4
y = 3

z = x > y

print(z)

print(class(z))

print(typeof(z))

Complex Data type in R

R supports complex data types that are set of all the complex numbers. The complex data type is to store numbers with an imaginary component.

Operators in R

R provides various operators for performing arithmetic, logical, and relational operations. Understanding these operators is essential for data manipulation and analysis.

  • Arithmetic Operators: +, -, *, /, ^ (addition, subtraction, multiplication, division, exponentiation)
  • Relational Operators: ==, !=, <, >, <=, >= (equal to, not equal to, less than, greater than, less than or equal to, greater than or equal to)
  • Logical Operators: &&, ||, ! (logical AND, logical OR, logical NOT)
  • Assignment Operator: <- (assigns a value to a variable)
            # Example of arithmetic and relational operators
            a <- 10
            b <- 5
            sum <- a + b
            is_greater <- a > b
            print(sum)
            print(is_greater)

[1] 15
[1] TRUE

Data Structures in R

A data structure is a particular way of organizing data in a computer so that it can be used effectively. The idea is to reduce the space and time complexities of different tasks. Data structures in R programming are tools for holding multiple values.

  • vectors
  • Matrices
  • Array

R’s base data structures are often organized by their dimensionality (1D, 2D or nD) and whether they’re homogeneous (all elements must be of the identical type) or heterogeneous (the elements are often of various types). This gives rise to the six data types which are most frequently utilized in data analysis.

vectors

A vector is an ordered collection of basic data types of a given length. The only key thing here is all the elements of a vector must be of the identical data type e.g homogeneous data structures. Vectors are one-dimensional data structures.

OR

R Vectors are the same as the arrays in R language which are used to hold multiple data values of the same type. One major key point is that in R Programming Language the indexing of the vector will start from '1' and not from '0'. We can create numeric vectors and character vectors as well.

Creating a vector in R

A vector is a basic data structure that represents a one-dimensional array. to create a array we use the "c" function which the most common method use in R Programming Language. We can also use seq() function or use colons ":" also as shown in the example. 

X<- c(61, 4, 21, 67, 89, 2)
cat('using c function', X, '\n')

Y<- seq(1, 10, length.out = 5) 
cat('using seq() function', Y, '\n') 

Z<- 2:7
cat('using colon', Z)
using c function 61 4 21 67 89 2 using seq() function 1 3.25 5.5 7.75 10 using colon 2 3 4 5 6 7

Types of R vectors

  • Numeric vectors
  • Character vectors
  • Logical vectors

Numeric vectors

Numeric vectors are those which contain numeric values such as integer, float, etc. The L suffix in R is used to specify that a number is an integer and not a numeric (floating-point) value. 

v1 <- c(4, 5, 6, 7)
typeof(v1)

v2 <- c(1L, 4L, 2L, 5L)
typeof(v2)
[1] "double" [1] "integer"

Character vectors

Character vectors in R contain alphanumeric values and special characters. In R, when a vector contains elements of mixed types (like characters and numbers), R automatically coerces the entire vector to a single type. Since characters are more general than numbers in R, the vector is coerced to a character vector. 

v1 <- c('geeks', '2', 'hello', 57)
typeof(v1)
[1] "character"

Logical vectors

Logical vectors in R contain Boolean values such as TRUE, FALSE and NA for Null values. In R, NA is a special value used to represent missing or undefined data. When used in a logical vector, NA is treated as a logical value because it is specifically designed to work with logical vectors and other types of data. 

v1 <- c(TRUE, FALSE, TRUE, NA)
typeof(v1)
[1] "logical"

Accessing R vector elements

Accessing elements in a vector is the process of performing operation on an individual element of a vector. There are many ways through which we can access the elements of the vector. The most common is using the '[]', symbol.

X <- c(2, 5, 18, 1, 12)

Y <- c(4, 8, 2, 1, 17)

Array

Arrays are important data storage structures defined by a fixed number of dimensions. Arrays are used for the allocation of space at contiguous memory locations.

Arrays are used to store data in a tabular form. It is a collection of elements of the same data type. The elements of an array are stored in contiguous memory locations. An array can have one or more dimensions.

Creating an Array

We can create an array in R using the array() function. The array() function takes a vector of data and dimensions as arguments.

# Creating a 2D array
my_array <- array(1:12, dim = c(3, 4))
print(my_array)

Output:

[,1] [,2] [,3] [,4]
[1,] 1 4 7 10
[2,] 2 5 8 11
[3,] 3 6 9 12

Types of Arrays

  • Uni-dimensional Array
  • Multi-dimensional Array
  • Array using vectors

Uni-dimensional Array

A one-dimensional array is a linear array that contains elements in a single row or column. It can be created using the array() function with a single dimension.

# Creating a uni-dimensional array
uni_array <- array(1:5, dim = c(5))
print(uni_array)

Output:

[1] 1 2 3 4 5

Multi-dimensional Array

A multi-dimensional array is an array with more than one dimension, such as a 2D or 3D array. It can be created using the array() function with multiple dimensions.

# Creating a multi-dimensional array (2D)
multi_array <- array(1:12, dim = c(3, 4))
print(multi_array)

Output:

[,1] [,2] [,3] [,4]
[1,] 1 4 7 10
[2,] 2 5 8 11
[3,] 3 6 9 12

Array using vectors

We can create an array using vectors by specifying the dimensions. The array will be filled with the values from the vector in a column-wise manner.

# Creating an array using vectors

Control Flow Statements in R

R provides control flow statements like if-else, for loops, while loops, and repeat loops to control the execution of code based on conditions.

  • if condition
  • if-else condition
  • for loop
  • while loop
  • repeat loop
  • break statement
  • next statement

if condition

This control structure checks the expression provided in parenthesis is true or not. If true, the execution of the statements in braces {} continues.

syntax: 
            if (condition) {
                # statements to execute if condition is true
            }

Example:

            x = 10
            if (x > 5) {
                print("x is greater than 5")
            }
[1] "x is greater than 5"

if-else condition

It is similar to if condition but when the test expression in if condition fails, then statements in else condition are executed.

syntax: 
            if (condition) {
                # statements to execute if condition is true
            } else {
                # statements to execute if condition is false
            }

Example:

            x = 3
            if (x > 5) {
                print("x is greater than 5")
            } else {
                print("x is not greater than 5")
            }
[1] "x is not greater than 5"

for loop

It is used to iterate over a sequence (like a vector or list) or a range of numbers. The loop executes a block of code for each element in the sequence.

syntax: 
            for (variable in sequence) {
                # statements to execute for each element
            }

Example:

            for (i in 1:5) {
                print(paste("Iteration:", i))
            }
[1] "Iteration: 1" [1] "Iteration: 2" [1] "Iteration: 3" [1] "Iteration: 4" [1] "Iteration: 5"

while loop

It is used to repeatedly execute a block of code as long as a specified condition is true.

syntax: 
            while (condition) {
                # statements to execute while condition is true
            }

Example:

            count <- 1
            while (count <= 5) {
                print(paste("Count:", count))
                count <- count + 1
            }
[1] "Count: 1" [1] "Count: 2" [1] "Count: 3" [1] "Count: 4" [1] "Count: 5"

repeat loop

It is used to repeatedly execute a block of code until a break statement is encountered.

syntax: 
            repeat {
                # statements to execute
                if (condition) {
                    break  # exit the loop
                }
            }

Example:

            count <- 1
            repeat {
                print(paste("Count:", count))
                count <- count + 1
                if (count > 5) {
                    break  # exit the loop when count exceeds 5
                }
            }
[1] "Count: 1" [1] "Count: 2" [1] "Count: 3" [1] "Count: 4" [1] "Count: 5"

break statement

It is used to exit a loop prematurely when a certain condition is met.

syntax: 
            for (variable in sequence) {
                if (condition) {
                    break  # exit the loop
                }
                # other statements
            }

Example:

            for (i in 1:10) {
                if (i == 6) {
                    break  # exit the loop when i equals 6
                }
                print(paste("Iteration:", i))
            }
[1] "Iteration: 1" [1] "Iteration: 2" [1] "Iteration: 3" [1] "Iteration: 4" [1] "Iteration: 5"

next statement

It is used to skip the current iteration of a loop and move to the next iteration.

syntax: 
            for (variable in sequence) {
                if (condition) {
                    next  # skip to the next iteration
                }
                # other statements
            }

Example:

            for (i in 1:10) {
                if (i %% 2 == 0) {
                    next  # skip even numbers
                }
                print(paste("Odd number:", i))
            }
[1] "Odd number: 1" [1] "Odd number: 3" [1] "Odd number: 5" [1] "Odd number: 7" [1] "Odd number: 9"
